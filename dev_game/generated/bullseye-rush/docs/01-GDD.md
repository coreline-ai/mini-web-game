# 01 · GDD — Bullseye Rush

## 한 줄 피치
조준선은 스스로 움직인다 — 완벽한 순간에 탭해서 불스아이를 꿰뚫고 12라운드를 클리어하라.

## 코어 루프 (30초)
1. 조준 마커가 좌우로 자동 왕복한다 (라운드마다 빨라짐).
2. **탭 = 그 순간의 위치에서 화살 수직 발사** — 조준 자체가 타이밍이다.
3. 과녁 명중 +30. **불스아이(중심 34% 반경) 명중 시 콤보 x2→x3→x4** 로 +60/+120/+180/+240.
4. 라운드의 모든 과녁을 제한 화살로 파괴 → CLEAR (+100 + 남은 화살×20) → 다음 라운드.
5. 보너스 별을 쏘면 화살 +1. 빗나가면 MISS — 콤보 리셋.
6. **미스 3회 또는 화살 소진 = 게임 오버.** 12라운드 전부 클리어 = 올클리어.

## 조작
**탭 한 번이 전부.** 드래그 없음, 이동 없음 — 왕복하는 조준선을 읽고 최적의 순간을 고르는 순수 타이밍 게임.

## 점수
| 행동 | 점수 |
|---|---|
| 과녁 명중 | +30 |
| 불스아이 | +60 × 콤보(최대 x4) |
| 라운드 클리어 | +100 + 남은 화살 ×20 |
| 보너스 별 | +50 + 화살 +1 |
| 생존 점수 | 없음 (정확도만이 자원) |

## 난이도 곡선 (라운드 테이블)
- 과녁 수: R1-2 한 개 → R3-4 두 개 → R5+ 세 개
- 스윕 속도: 240px/s + 라운드당 +26 (R12 ≈ 526px/s)
- R3+: 과녁 좌우 이동 시작(+14/라운드), R5+: 상하 바운스 추가, 과녁 크기 라운드당 -3.5%(최소 72%)
- 화살 수: 과녁×2+3에서 라운드가 오르면 여유분 점감 → 후반은 미스 허용치가 급감

## 실패/재시도/승리
- 미스 3회 or 화살 소진+과녁 잔존 → 게임오버(점수/최고점) → RETRY
- R12 클리어 = 올클리어(WIN 플래시) → 최종 점수 정산

## 콘텐츠
사격장 배경 3종(초원 연습장/석양 토너먼트/등불 야시장 — 라운드 진행 크로스페이드) · 불스아이 과녁 / 보너스 별 / 화살 · HUD(점수·라운드·ARROWS·MISS·COMBO 배지·ROUND 배너) · SFX 5종+BGM

## 기존 게임과의 차별점
Sky Archer(드래그 이동+자동발사)·Castle Archer(드래그 조준+수동발사·성문HP)와 달리 **조준이 자동이고 입력은 타이밍 탭뿐**이다. 스포너/낙하물이 아예 없고(라운드제 과녁 패턴), 실패 자원도 시간·HP가 아니라 **정확도(미스·화살)** 다. 입력·루프·실패 구조 모두 신규.
