# 05 · Adversarial Review — Jungle Arc Shot

## "리스킨 아니냐"에 대한 반박
1. **발사체 물리가 처음 존재**: 본 레포 4작 중 유일하게 발사체가 중력(950)+바람 가속을 받으며, 궤적 미리보기가 동일 해석해로 그려진다. 직선 화살(Castle/Bullseye)과 판정·조준 문법이 근본적으로 다르다.
2. **Spawner 미사용**: 낙하물 스폰이 없다. JungleRoundSystem의 클러스터 배치가 콘텐츠를 만든다.
3. **관통 콤보 = 고유 점수 축**: 화살이 표적에서 멈추지 않고 n번째 관통 ×n 배점 — 다른 3작에 없는 메커니즘이며 "화살 = 표적 절반+1" 경제가 이를 강제한다.
4. **바람 = 라운드 변수**: 매 라운드 방향이 교대되고 세기가 증가해 같은 배치도 조준이 달라진다. HUD에 방향/세기를 노출해 "읽기" 스킬을 만든다.
5. **입력 문법 신규**: 새총식(당김 반대 발사)은 드래그-조준(Castle: 가리키는 방향)과 반대 벡터 + 파워 개념 추가.

## 남는 리스크 (정직)
- Foundation shell(씬/HUD/저장/오디오/스테이지)은 설계 의도대로 재사용.
- 승리(올클리어)와 패배가 같은 GameOver 화면으로 정산 — 결과 화면 분기는 확장 항목.
- 풍선 상승 드리프트의 "아래 재진입"은 단순 랩 — 자연스러운 페이드 재진입은 폴리시 후보.
