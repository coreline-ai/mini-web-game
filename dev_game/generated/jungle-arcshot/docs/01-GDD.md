# 01 · GDD — Jungle Arc Shot

## 한 줄 피치
바람을 읽고 포물선을 구부려라 — 한 발로 여러 과일을 꿰뚫는 정글 트릭샷 사격.

## 코어 루프 (30초)
1. 라운드 시작: 과일·풍선 클러스터 + 이번 라운드의 바람(방향/세기) 표시.
2. 드래그(새총식: 당길수록 파워, 반대 방향 발사) → **중력+바람이 반영된 점선 궤적 미리보기**.
3. 놓으면 화살이 포물선으로 날아감 — 표적을 **관통**하며 n번째 표적 = 기본점 ×n ("PIERCE xN!").
4. 화살은 표적 수의 절반+1만 지급 — **관통 없이는 클리어 불가능한 경제**.
5. 전 표적 제거 → CLEAR(+120+잔여 화살×25) → 다음 라운드(바람 강해지고 방향 교대).
6. 화살 소진+표적 잔존 = 게임오버. **R10 전체 클리어 = 올클리어 승리.**

## 조작
드래그 = 각도+파워(새총), 미리보기 궤적 확인 → 놓기 = 발사. 이동/조준선/타이밍 없음 — **탄도 계산**이 스킬이다.

## 점수
| 행동 | 점수 |
|---|---|
| 과일 (1번째 관통) | +40 |
| 풍선 (1번째 관통) | +25 (+코인 1) |
| n번째 관통 | 기본점 ×n (2번째 x2, 3번째 x3…) |
| 라운드 클리어 | +120 + 잔여 화살 ×25 |
| 생존 점수 | 없음 |

## 난이도 곡선
- 표적: 과일 2→7 + 풍선 0→4 (세로 줄 클러스터 = 관통 라인 유도)
- 바람: R2부터 ±(30+16n) px/s², **라운드마다 방향 교대** — 매 라운드 조준 재학습
- R4+: 풍선이 위로 드리프트(상단 이탈 시 아래 재진입) — 움직이는 표적 리드샷
- 화살: ⌈표적/2⌉+1 — 후반은 평균 관통 2 이상을 강제

## 실패/재시도/승리
화살 소진(비행 중 화살 소멸 후) + 표적 잔존 → 게임오버 → RETRY. R10 클리어 = 올클리어(그린 플래시).

## 콘텐츠
정글 배경 3종(새벽 캐노피/한낮 폭포/반딧불 황혼) · 과일/풍선/대나무 화살 · HUD(점수·ROUND n/10·ARROWS·WIND 방향/세기·일시정지) · SFX 5종+BGM

## 기존 게임과의 차별점
Sky(이동+자동발사)·Castle(직선 저격+거점HP)·Bullseye(타이밍 탭)와 달리 **발사체가 물리(중력+바람)를 받고**, 핵심 스킬은 반응속도가 아닌 **탄도 예측**이다. 관통 콤보라는 고유 점수 축과 "화살 경제" 실패 구조도 처음.
