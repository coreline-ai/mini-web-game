# 01 · GDD — Castle Archer

## 한 줄 피치
성벽 위 왕실 궁수가 되어, 성문이 뚫리기 전에 진군하는 고블린 습격대를 드래그 조준 저격으로 전멸시켜라.

## 코어 루프 (30초)
1. 고블린이 화면 위에서 성벽(하단)을 향해 진군한다.
2. 드래그로 조준선을 맞추고 → 놓으면 화살 발사 (수동 저격).
3. 명중 +30, **정조준 크리티컬(수평 중심 ±16%) 2배 +60**, 방패병은 첫 발에 방패 파괴 → 두 번째 발로 처치.
4. 놓친 고블린이 성벽 라인에 도달하면 **성문 HP -1** (하트 3개).
5. 치유 물약을 쏘거나 받으면 성문 HP +1 회복 (최대 3).
6. 웨이브가 오를수록 빠르고 촘촘해진다 → 성문 HP 0 = 함락(게임 오버).

## 조작
- **드래그 = 조준** (점선 조준선 + 레티클, 위쪽 반구 클램프) / **놓기 = 발사** (쿨다운 170ms)
- 짧은 탭도 그 방향 즉시 발사. 궁수는 성벽 중앙 고정 — 이동 게임이 아니라 **조준 게임**.

## 점수
| 행동 | 점수 |
|---|---|
| 고블린 처치 | +30 |
| 크리티컬 처치 (정조준) | +60 (×2) |
| 방패 파괴 | +10 |
| 치유 물약 | +50 + 성문 HP 1 |
| 웨이브 보너스 | +100×(웨이브-1) |
| 생존 | +1/초 |

## 난이도 곡선
- 웨이브(=레벨) 18초마다 상승. 스폰 1150ms→가속, 진군 속도 70→150.
- 1분(웨이브 3~4): 한 발씩 조준할 여유. 5분(웨이브 12+): 다수 동시 진군+방패병 혼합 — 우선순위 판단 강제.
- 매 5번째 고블린 = 방패병(2발): 화살 배분 vs 위협 순위의 전술 선택.

## 실패/재시도
성문 HP 3 소진 → 함락 연출 → GameOver(점수/최고점/코인) → RETRY. 원힛 사망이 아니라 거점 HP제 — 물약으로 실수 만회 가능.

## 콘텐츠
배경 3종(새벽 초원/석양 공성/횃불 밤, 웨이브 크로스페이드) · 고블린/방패 고블린(2HP)/치유 물약 · 화살(풀 24) · HUD(점수·웨이브·성문 하트 3·일시정지·WAVE 배너) · SFX 5종+BGM.

## 기존 게임과의 차별점
dodge류(똥 피하기·meteor-dash)는 "이동해서 피한다", sky-archer는 "이동 조준+자동 발사". **Castle Archer는 이동이 없다** — 드래그가 곧 조준, 발사는 전부 수동, 실패는 충돌 원힛이 아닌 **거점(성문) HP**. 입력·판정·실패 구조가 모두 다른 방어형 저격 루프다.
