# 에셋·오디오 계획

## 아트 디렉션

고품질 2.5D 스타일라이즈드 리얼리즘. 상단에서 내려다보는 3/4 탑다운 카메라를 고정한다. 인물은 배경보다 명도 대비가 높고, 검은 감염 체액에는 청록 반사광을 넣어 붉은 고어 없이도 강한 충격을 전달한다.

## 신규 전용 이미지

| 역할 | 최소 산출물 | 모션 요구 |
|---|---|---|
| 주인공 | 상단을 보는 후면 대기·이동·발사·피격 시트 | 8~12fps, 얼굴이 보이지 않는 후면 시점, 하체 보행과 상체 조준 분리감 |
| 워커 | 이동·공격·사망 시트 | 불균일 보폭과 팔 후속 동작 |
| 러너/크롤러 | 이동·돌진 시트 | 낮은 중심과 빠른 실루엣 변화 |
| 브루트 | 이동·내려찍기 시트 | 무거운 예비 동작과 후속 흔들림 |
| 타이탄 | 이동·포효·공격 시트 | 큰 체중 이동과 약점 노출 |
| 무기 | 5종 아이콘·총열 레이어 | 회전, 반동, 충전 발광 |
| 배경 | 새벽·낮·황혼·밤·블러드문 | 동일 지형 구도, 조명만 자연스럽게 전환 |
| FX/UI | 총구 화염·체액·폭발·카드·패널 | 알파 PNG, 텍스트 없는 프레임 |

## 기술 규칙

- 모든 이미지가 `assets/**` 아래에 자체 포함된다.
- 런타임 핵심 이미지는 SVG 플레이스홀더를 사용하지 않는다.
- 스프라이트는 동일 셀, 동일 루트 앵커, 동일 바닥선, 프레임 간 불투명 픽셀 비접촉을 지킨다.
- 주인공 몸체와 5종 무기는 분리 레이어로 제작한다. 몸체는 상단을 보는 후면 시점으로 고정하고 모든 무기 총구도 위쪽을 기준 방향으로 삼는다.
- 생성 원본은 GPT Image 2 built-in imagegen을 사용하고 `codex-gpt-imagegen-skill` provenance를 기록한다.
- 투명 에셋은 평면 크로마 키 생성 후 공식 제거 도구로 알파 변환하고 모서리·잔색을 검사한다.
- 배경은 최종 1440×2560 이상으로 전달하며 비율을 늘려 찌그러뜨리지 않는다.

## 오디오

- BGM: 시간대에 따라 레이어가 증감하는 저주파 산업 앰비언트.
- 기관포: 총열 기동, 연속 발사, 과열 경고, 냉각 배기음을 분리한다.
- 특수 무기: 산탄 충격, 전기 연쇄, 로켓 발사·폭발, 레일 충전·방출.
- 적: 역할별 발소리·호흡·포효·피격·사망.
- UI: 무기 전환, 충전 완료, 강화 구매, 무전 호출.
- 음소거, 백그라운드 정지, 장면 전환 정지를 보장한다.
