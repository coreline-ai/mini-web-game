# 적대적 검토

## 단순 리스킨이 아닌 이유

기초 생성기의 낙하 장애물 회피 루프는 제거한다. 이 게임은 목표 추적, 자동·수동 조준 전환, 탄환 풀, 피해·체력, 무기 열과 충전, 군집 조향, 원거리 공격, 보스 상태 머신, 영구 성장으로 구성된 별도 전투 시뮬레이션이다.

## 실패하기 쉬운 지점

| 위험 | 방지 계약 |
|---|---|
| 좀비 수만 늘고 행동은 동일함 | 역할별 속도·크기·조향·공격 상태를 분리한다. |
| 자동 조준 때문에 플레이어가 할 일이 없음 | 수동 드래그가 이동과 조준을 동시에 바꾸고 위험 표적 우선순위를 뒤집게 한다. |
| 과열이 단순 대기만 강요함 | 다른 충전 무기로 즉시 교대해 공격 리듬을 유지한다. |
| 2K 캔버스가 저사양 모바일에서 과열됨 | 풀링, 활성 수 상한, 품질 단계, 30fps 폴백을 적용한다. |
| 스프라이트가 정지 컷처럼 보임 | 걷기 위상 분산, 상하 흔들림, 회전, 경직, 후속 동작을 코드와 프레임 애니메이션 양쪽에서 결합한다. |
| 낮·밤이 색 필터뿐임 | 배경·조명·스폰 조합·시야·무전 이벤트를 함께 변경한다. |
| 영구 강화가 첫 판을 망가뜨림 | 초기 레벨로도 2분 생존 가능하며 강화 비용은 초반 1~2판에 첫 구매가 가능하도록 조정한다. |

## 완료 거부 조건

- 핵심 플레이어·좀비·보스가 SVG 또는 단일 정지 플레이스홀더인 경우.
- 실제 총탄과 피해 없이 점수만 증가하는 경우.
- 자동 조준 또는 드래그 조준 중 하나가 동작하지 않는 경우.
- 특수 무기 충전이 시간과 처치 양쪽에서 검증되지 않은 경우.
- 영구 강화가 다음 세션에 반영되지 않는 경우.
- 모바일 세 가지 뷰포트 또는 production-demo gate가 실패한 경우.
