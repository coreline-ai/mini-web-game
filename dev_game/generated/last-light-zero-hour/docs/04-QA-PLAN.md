# QA 계획

## 기능 검증

- PLAY가 Game 장면으로 진입한다.
- 무입력 상태에서 가장 가까운 감염체를 자동 조준하고 실제 HP를 감소시킨다.
- 하단 드래그가 주인공을 전투 구역 안에서 이동시키고 조준 방향을 변경한다.
- 기관포 열이 발사 중 상승하고 과열 시 정지한 뒤 냉각 후 복구된다.
- 특수 무기는 시간과 처치 양쪽으로 충전된다.
- 기관포 자동사격 중 각 특수무기 카드 탭이 현재 조준 방향으로 즉시 발동하며, 서로의 발사 쿨다운을 막지 않는다.
- 상단 자동 사냥 버튼이 ON/OFF를 명확히 표시하며, ON 상태에서 주인공이 안전 레인으로 이동하고 준비된 특수무기를 위협도에 따라 사용한다.
- 작은 러너, 일반 워커, 대형 브루트, 타이탄의 크기·속도·행동이 구분된다.
- 시간이 흐를수록 화면 적 수와 조합이 증가한다.
- 낮·밤 팔레트와 배경이 중단 없이 전환된다.
- 사망 결과가 영구 자원에 반영되고 무기고 구매가 다음 판 수치에 적용된다.

## 모션·시각 검증

- 각 시트 프레임 수, 셀 피치, 기준선, 알파 경계를 검사한다.
- 같은 종류 좀비가 완전히 동일한 위상으로 걷지 않도록 시작 프레임을 분산한다.
- 총열 회전, 총구 화염, 탄피, 반동이 발사율과 동기화된다.
- 피격 방향으로 경직하고 검은 체액 FX가 충돌점에서 발생한다.
- 390×844, 430×932, 1080×1920에서 HUD·무기 카드·조작 구역이 겹치지 않는다.

## 장시간·성능 검증

- 10분 가속 시뮬레이션 후 스폰율, 충전, 날짜 증가가 유한값을 유지한다.
- 활성 객체 수가 정의된 상한을 넘지 않는다.
- Pause/Home/GameOver에서 타이머·오디오·입력이 누적되지 않는다.
- 재도전 5회 후 이벤트 리스너와 물리 그룹이 중복되지 않는다.
