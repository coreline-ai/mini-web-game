# Parcel Sort Rush — GDD

## 한줄 피치

택배 공장 컨베이어로 밀려오는 박스를 한 손 드래그로 올바른 분류 슈트에 보내며, 점점 빨라지는 물량 폭주를 버티는 모바일 세로형 분류 아케이드.

## 제작 판단

- Decision: `custom-loop`
- 이유: 핵심 재미가 “위에서 떨어지는 물체 회피”가 아니라 “라벨을 읽고 올바른 목적지로 분류”하는 판단/정확도 게임이다.
- Archetype 사용: 공통 Phaser/Vite Foundation만 사용하고 gameplay는 택배 분류 전용 시스템으로 구현한다.

## 30초 코어 루프

1. 컨베이어 위에 택배가 등장한다.
2. 택배 라벨/색/아이콘을 확인한다.
3. 손가락으로 택배를 끌어 올바른 분류 슈트에 놓는다.
4. 맞으면 점수/콤보/사운드가 즉시 나온다.
5. 틀리거나 놓치면 라이프가 감소한다.
6. 속도와 물량이 증가해 “한 판 더”를 유도한다.

## 조작

- 모바일/마우스 공통: 택배를 누르고 드래그한 뒤 슈트 위에서 놓기
- 한 손 플레이 기준: 하단 4개 슈트와 중앙 컨베이어를 엄지 범위 안에 배치

## 분류 단위

| id | 라벨 | 의미 |
|---|---|---|
| `north` | A권역 | 수도권/북부 분류 |
| `south` | B권역 | 남부/지방 분류 |
| `cold` | 냉장 | 냉장/신선 배송 |
| `fragile` | 취급주의 | 파손주의/주의 배송 |

## 점수/실패

- 정분류: +100점
- 콤보 8개마다 보너스
- 10연속 정분류: PERFECT SORT 보너스
- 오분류: 라이프 -1
- 게이트 통과 후 미분류: 라이프 -1
- 라이프 0: Game Over

## 난이도

- 1분: 택배 1개씩, 느린 속도, 색/라벨 판독 쉬움
- 3분: 속도 증가, 화면에 2~4개 동시 등장
- 5분: 러시 이벤트, 스폰 간격 감소, 판단 압박 증가

## MVP 포함

- Boot/Loading/Home/Game/Pause/GameOver
- 4분류 슈트
- 드래그 분류
- 정분류/오분류/미분류 판정
- 점수/콤보/라이프/스테이지
- 러시 이벤트
- 게임 전용 production PNG 에셋 + OGG 오디오
- 3종 1080x1920 택배센터 배경
- 에셋 provenance/독립성 manifest

## MVP 제외

- 실제 송장 OCR
- 로그인/서버 랭킹
- 광고/IAP
- 물류센터 경영 메타
