# Sky Archer

Generated by `dev_game/generator` from the `sky-archer` spec.

## Run

```bash
npm install
npm run dev
npm run build
```

## Scope

This starter intentionally targets a small mobile portrait arcade game:

- Boot/Loading/Home/Game/Pause/GameOver scenes
- One-hand drag control
- Falling hazards and collectibles
- Score, best score, pause, audio, localStorage
- Placeholder images and procedural audio

Excluded by design: backend, ads, payments, multiplayer, heavy liveops, native packaging.
