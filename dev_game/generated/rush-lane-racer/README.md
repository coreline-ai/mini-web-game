# Rush Lane Racer

Generated by `dev_game/generator` from `rush-lane-racer.spec.json`, then expanded into a racing-focused playable starter.

## Concept

One-hand vertical arcade racing survival: drag the player car left/right, dodge traffic, collect coins and powerups, and survive escalating warning speed zones.

## Included runtime scope

- Phaser 3 + Vite mobile portrait canvas, 1080×1920 logical size
- Required scenes: Boot, Loading, Home, Game, Pause, GameOver
- Drag-only one-hand control
- Animated scrolling road, lane stripes, city backdrop
- Traffic hazards: car, truck, motorcycle, oil, cone, roadblock, police, fake coin, boss truck
- Collectibles/powerups: coin, nitro, shield, magnet, slow, repair
- Survival score, dodge score, near-miss bonus, combo, coins, local best
- 30-second warning/speed-zone event and endless difficulty ramp
- Procedural WAV starter SFX/BGM from the dev_game factory

## Run

```bash
npm install
npm run dev
npm run build
```

## Factory note

The base factory still validates the core supported spec fields. Racing-specific catalogs are stored as extra JSON metadata and consumed by this generated starter.
