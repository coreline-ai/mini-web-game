# 01 · GDD — Rush Lane Racer

Rush Lane Racer는 한 손 드래그로 차량을 좌우 이동시키며 교통 차량, 트럭, 오토바이, 오일, 공사 콘, 도로 블록, 가짜 코인, 보스 트럭을 피하고 코인/니트로/방패/자석/슬로우/수리 키트를 획득하는 세로형 모바일 아케이드 레이싱 게임이다. 목표는 1분 안에는 즉시 이해되고, 5분 이후에는 스폰 밀도와 속도가 급상승하는 반복 플레이 경험이다.

핵심 루프는 Home → PLAY → 도로 주행 → 회피/수집 → 니어미스/콤보 → 충돌 → GameOver → 재시도이다. 조작은 플레이어 차량의 x축 드래그만 사용하며, 차량은 하단 20% 영역에서 빠르게 목표 x로 따라간다. 점수는 생존 시간, 회피, 니어미스, 코인 수집으로 증가한다. 스테이지는 도시 낮, 네온 밤, 사막 석양 배경으로 시각적 진행감을 제공하고, 이후 반복 순환과 난이도 상승으로 Endless 감각을 만든다. 실패 조건은 보호막/무적이 없을 때 장애물과 충돌하는 것이다.
