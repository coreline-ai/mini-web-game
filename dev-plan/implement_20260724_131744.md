기존 GitHub Pages 게임을 유지하면서 LAST LIGHT: 제로 아워를 프로젝트 하위 경로로 재현 가능하게 빌드·배포하고, 배포 후 실제 플레이까지 검증한다.

- 기존 게임을 `https://coreline-ai.github.io/mini-web-game/`에 계속 배포한다.

## 문서 기반 제약 사항

- Pages는 정적 artifact만 제공하므로 현재 게임의 `localStorage` 저장 범위는 그대로 유지한다.
- Pages 배포 workflow는 저장소의 단일 source of truth로 유지하며, 같은 artifact를 배포하는 두 번째 workflow를 만들지 않는다.
- Last Light의 전투 에셋은 `asset-manifest.json`과 runtime delivery plugin의 allowlist를 기준으로 배포한다.
- 현재 target QA가 통과한 390×844, 430×932, 1080×1920 viewport 계약을 유지한다.
